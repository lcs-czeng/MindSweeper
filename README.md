# MindSweeper

A study and focus tool for iOS built with SwiftUI.

## Getting Started

1. Clone the repository
2. Open `MindSweeper.xcodeproj` in Xcode
3. Select a simulator or your device
4. Press ▶ to build and run

## Project Structure

```
MindSweeper/
├── MindSweeperApp.swift     # App entry point
├── ContentView.swift        # Root view with tab navigation
└── Assets.xcassets/         # App icons, colors, and images
```

## Tech Stack

- Swift
- SwiftUI
- iOS 17+
